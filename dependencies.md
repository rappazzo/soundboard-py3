# Dependencies

Slack:

    pip3 install slackclient
    pip3 install nest_asyncio

Audio:

    pip3 install pydub

  - requires: `brew install ffmpeg` or `apt-get install ffmpeg`

Library Monitoring:

    pip3 install watchdog

REST API:

    pip3 install connexion 'connexion[swagger-ui]' python_dateutil typing
